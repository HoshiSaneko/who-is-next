## 🚀 CDN 上传步骤

### 方案 1: 使用阿里云 OSS

```bash
# 安装 ossutil
# https://help.aliyun.com/document_detail/120075.html

# 配置 OSS
ossutil config

# 批量上传 WebP 文件
ossutil cp -r public/covers/*.webp oss://your-bucket/covers/
ossutil cp -r public/avatars/*.webp oss://your-bucket/avatars/
ossutil cp -r public/up-members/*.webp oss://your-bucket/up-members/
ossutil cp -r public/images/*.webp oss://your-bucket/images/
```

### 方案 2: 使用七牛云

```bash
# 安装 qshell
# https://developer.qiniu.com/kodo/tools/1302/qshell

# 配置七牛云
qshell account <AccessKey> <SecretKey> <Name>

# 批量上传
qshell qupload2 --src-dir=public/covers --bucket=your-bucket --key-prefix=covers/
qshell qupload2 --src-dir=public/avatars --bucket=your-bucket --key-prefix=avatars/
qshell qupload2 --src-dir=public/up-members --bucket=your-bucket --key-prefix=up-members/
qshell qupload2 --src-dir=public/images --bucket=your-bucket --key-prefix=images/
```

### 方案 3: 使用腾讯云 COS

```bash
# 安装 COSCMD
pip install coscmd

# 配置 COS
coscmd config -a <SecretId> -s <SecretKey> -b <BucketName> -r <Region>

# 批量上传
coscmd upload -r public/covers/ covers/
coscmd upload -r public/avatars/ avatars/
coscmd upload -r public/up-members/ up-members/
coscmd upload -r public/images/ images/
```

## 🔧 代码配置

上传完成后，在项目中配置 CDN 基础 URL：

### 创建 CDN 配置文件

**`src/config/cdn.ts`**
```typescript
export const CDN_BASE_URL = 'https://cdn.yourdomain.com'; // 替换为你的 CDN 域名

export function getCDNUrl(path: string): string {
  // 如果已经是完整 URL，直接返回
  if (path.startsWith('http')) {
    return path;
  }
  
  // 移除开头的斜杠
  const cleanPath = path.startsWith('/') ? path.slice(1) : path;
  
  return `${CDN_BASE_URL}/${cleanPath}`;
}
```

### 更新 OptimizedImage 组件

**`src/components/OptimizedImage.tsx`**
```tsx
import { useState, useEffect, useRef } from 'react';
import { getCDNUrl } from '@/src/config/cdn';

interface OptimizedImageProps {
  src: string;
  alt: string;
  className?: string;
  width?: number;
  height?: number;
  loading?: 'lazy' | 'eager';
  useCDN?: boolean; // 是否使用 CDN
}

export function OptimizedImage({
  src,
  alt,
  className = '',
  width,
  height,
  loading = 'lazy',
  useCDN = true // 默认使用 CDN
}: OptimizedImageProps) {
  const [imageSrc, setImageSrc] = useState<string>('');
  const [isLoaded, setIsLoaded] = useState(false);
  const imgRef = useRef<HTMLImageElement>(null);

  useEffect(() => {
    const baseSrc = useCDN ? getCDNUrl(src) : src;
    
    // Convert to WebP
    const webpSrc = baseSrc.replace(/\.(jpg|jpeg|png)$/i, '.webp');

    // Try WebP first
    const img = new Image();
    img.onload = () => {
      setImageSrc(webpSrc);
    };
    img.onerror = () => {
      // Fallback to original format
      setImageSrc(baseSrc);
    };
    img.src = webpSrc;
  }, [src, useCDN]);

  const handleLoad = () => {
    setIsLoaded(true);
  };

  if (!imageSrc) {
    return (
      <div
        className={`${className} bg-gray-200 animate-pulse`}
        style={{ width, height }}
      />
    );
  }

  return (
    <img
      ref={imgRef}
      src={imageSrc}
      alt={alt}
      className={`${className} transition-opacity duration-300 ${
        isLoaded ? 'opacity-100' : 'opacity-0'
      }`}
      width={width}
      height={height}
      loading={loading}
      onLoad={handleLoad}
      decoding="async"
    />
  );
}
```

### 使用示例

```tsx
import { OptimizedImage } from '@/src/components/OptimizedImage';

// 使用 CDN（默认）
<OptimizedImage 
  src="/covers/BV1AnJCztE1x.jpg" 
  alt="视频封面"
/>

// 不使用 CDN（本地加载）
<OptimizedImage 
  src="/covers/BV1AnJCztE1x.jpg" 
  alt="视频封面"

/>
```

## 📝 注意事项

1. **CDN 域名配置**: 上传完成后记得配置 CDN 域名的 CNAME 解析
2. **缓存策略**: 建议设置 30 天缓存：`Cache-Control: max-age=2592000`
3. **HTTPS**: 确保 CDN 支持 HTTPS
4. **跨域配置**: 如果 CDN 域名与网站域名不同，需要配置 CORS
5. **原始文件**: 上传 WebP 后，可以保留原始文件作为后备
6. **测试**: 上传后在开发环境测试所有图片是否正常加载
